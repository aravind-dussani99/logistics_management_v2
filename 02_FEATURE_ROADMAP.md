# Feature Roadmap

## Phase 1: Foundation & Master Data (Current Focus)
-   [x] **Server Setup**: Backend (Node.js) and Frontend (React/Vite) initialization.
-   [x] **Database Setup**: SQLite integration for local development.
-   [x] **Master Data Management**:
    -   [x] **Business Info**: CRUD operations, file uploads (address proof), auto-increment IDs (10000+).
    -   [x] **Business Types**: Management of business categories (Supplier, Customer, etc.).
    -   [ ] **Site Locations**: Management of pickup/drop-off locations.
    -   [ ] **Vehicle Info**: Management of fleet details.
    -   [ ] **Material Info**: Management of transported materials.

## Phase 2: Core Operations
-   [ ] **Daily Trips**: Entry and tracking of daily vehicle trips.
-   [ ] **Received Trips**: Verification and processing of completed trips.
-   [ ] **Advances**: Management of driver/vendor advances.
-   [ ] **Daily Expenses**: Tracking operational expenses.

## Phase 3: Reporting & Analytics
-   [ ] **Reports Module**: Comprehensive reports for trips, expenses, and profitability.
-   [ ] **Dashboard**: Visual charts and graphs for quick insights.

## Phase 4: Advanced Features
-   [ ] **Authentication**: Secure login and role-based access control (RBAC).
-   [ ] **Cloud Migration**: Migration from SQLite to PostgreSQL/CloudSQL.
-   [ ] **Mobile Optimization**: Enhanced mobile experience for drivers/supervisors.
